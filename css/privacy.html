Privacy policy

Our services collect and store user data to remember that games and users exist. Examples of data collected include, but are not limited to:

  • Usernames
  • Account emails
  • Display names
  • User-submitted bios
  • The games you created
  • The games you joined
  • Game names
  • User-submitted game descriptions
  • Game passwords (not encrypted)
  • Whether you were killed in a game

We also collect user passwords, but we salt and encrypt them before storing them.

Only usernames, user display names, user bios, game names, game descriptions, the games you created and joined, and other game-related personal information are publicized.

User information is stored in an unencrypted lowdb database without backups on a laptop. This means that someone with access to our database will be able to access a few other data such as everyone's encrypted passwords. We are not responsible for spontaneous data loss. We do not sell your user information because we don't know to whom it should be sold.

If you would like to download or delete all your data, please email sy24484@pausd.us. This is not because we want to discourage users from doing this; rather, we simply have not implemented this yet.
